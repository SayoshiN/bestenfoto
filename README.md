# ♥ BestenFoto — Сайт голосования за лучшее фото

## 🚀 Быстрый старт

### 1. Установить зависимости
```bash
pip install flask bcrypt
```

### 2. Запустить сайт
```bash
# Только локально
python start.py

# Или с ngrok (публичная ссылка)
python start.py --ngrok
```

### 3. Открыть в браузере
```
http://localhost:5000
```

## 📷 Как добавить фото участников

**Способ 1 — через файловый менеджер:**
1. Скопируйте фото в папку `static/photos/`
2. Откройте http://localhost:5000/admin
3. Нажмите «📷 Фото» рядом с кандидатом и выберите файл

**Способ 2 — через панель администратора:**
1. Откройте http://localhost:5000/admin
2. В разделе «Добавить кандидата» заполните форму и прикрепите фото

Поддерживаемые форматы: JPG, PNG, WebP, GIF

## 🌐 Запуск через ngrok (публичный доступ)

### Установка ngrok:
1. Перейдите на https://ngrok.com/download
2. Скачайте и установите ngrok
3. Зарегистрируйтесь и получите токен: https://dashboard.ngrok.com
4. Настройте токен:
   ```bash
   ngrok config add-authtoken ВАШ_ТОКЕН
   ```
5. Запустите:
   ```bash
   python start.py --ngrok
   ```
6. В консоли появится публичная ссылка — поделитесь ей!

## 🗄 База данных

- Файл: `data/bestenfoto.db` (SQLite)
- Создаётся автоматически при первом запуске
- Таблицы: `users`, `candidates`, `votes`
- Все события выводятся в консоль в реальном времени

### Просмотр базы данных:
```bash
sqlite3 data/bestenfoto.db
.tables
SELECT * FROM users;
SELECT * FROM votes;
SELECT * FROM candidates;
.quit
```
